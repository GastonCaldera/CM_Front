
## Curiosity Cam App
This app uses the NASA Rover Mars API to fetch photos from the Curiosity, Perseverance, and Ingenuity rovers. The photos are displayed in a grid

## Getting Started
Install libraries

```bash
npm i
# or
yarn
```

Run App
```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```
